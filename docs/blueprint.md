# **App Name**: Hanoi Residences

## Core Features:

- Apartment Listings: Display a list of available apartments with key information such as price, location, and room type.
- Filtering and Search: Enable users to filter apartments based on district, price range, and room type.
- Apartment Details Page: Provide a detailed view of each apartment with images, address, price, design information, and other details.
- Admin Dashboard: Allow admin to login and manage apartment listings: add, edit, and delete apartments.
- Content Management: Enable the admin to input apartment details, including title, source code, room type, district, price, detailed information, exact address, and images.
- Data Storage: Use a database to store apartment listings and related information.
- AI-Powered Listing Summarization: AI tool which creates a listing summary for each apartment based on its features and amenities.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to create a sense of comfort and sophistication.
- Background color: Off-white (#FAF9F6) for a clean and spacious feel.
- Accent color: Muted gold (#C0B283) for highlights and calls to action.
- Headline font: 'Playfair', serif, for an elegant and high-end feel.
- Body font: 'PT Sans', sans-serif,  for clear and readable content. Font pairing with Playfair.
- Use minimalist icons to represent apartment features and amenities.
- Clean and spacious layout with a focus on visual hierarchy and ease of navigation.
- Subtle animations for transitions and interactions to enhance user engagement.
